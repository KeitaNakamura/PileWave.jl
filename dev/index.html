PileWave

Pile driving analysis based on the one-dimensional wave propagation using the finite element method

This program is mostly based on the study by Borja (1988).

How to run

Use REPL or Execution file (Windows).

REPL

Installation

pkg> registry add https://github.com/KeitaNakamura/KeitaNakamuraRegistry.git

pkg> add PileWave

Running simulations

julia> using PileWave

julia> PileWave.solve("input.toml")

Execution file (Windows)

Downloads

Download execution file from releases.

Running simulations

/path/to/PileWave.exe /path/to/input.toml

Input

The input TOML file should be the following structure.

[Input]
embedded_depth = 12.0 # m
gravity = 9.81 # m/s²
t_stop = 80e-3 # s
input_load = "load.csv" # specify the input file

[Output]
# directory = "case 1" # "<filename>.tmp" by default
# num_data = 200 # number of output data (200 by default)
history_points = [0.0, 14.25, 28.0] # coordinates to output time history data (0.0 means pile head)

[Advanced]
# shape = @jl Line3() # shape function (Line3() by default)
# CFL = 0.5 # Courant number (0.5 by default)

[[Pile]]
length = 27.7 # m
num_elements = 100
youngs_modulus = 210000e6 # N/m²
density = 7850.0 # kg/m³
area = 0.2139 # m²
perimeter = 7226e-3 # m

[[Pile]]
length = 0.3
num_elements = 5
youngs_modulus = 210000e6
density = 7850.0
area = 0.2792
perimeter = 7282e-3
area_bottom = 4.22 # π*(2.318/2)^2

[[SoilLayer]] # gravel
thickness = 9.05
shaft = {quake=1e-3, yield_stress=0.0133e6, damping=10640.0}

[[SoilLayer]] # clay
thickness = 0.65
shaft = {quake=1e-3, yield_stress=0.00213e6, damping=1704.0}

[[SoilLayer]] # gravel
thickness = 2.3
shaft = {quake=1e-3, yield_stress=0.07e6, damping=56000.0}
bottom = {quake=1e-3, yield_stress=1.403e6, yield_factor=0.1, damping=280600.0}

The above example file is here.

Input load data

load.csv must have time and load columns such as

time,load
0.0,0.0
0.1,239.00
0.2,478.00
[...]

where the intermediate values are linearly interpolated.

Parameters for soil model

The values in bracket represent the default values.

  • quake: $u_\mathrm{q}$ (inf: slip)
  • yield_stress: $R_\mathrm{u}$ (0.0: slip)
  • yield_factor: $\alpha$ (1.0 for shaft and 0.0 for bottom)
  • damping: $C$ (0.0)
Info

If you want to impose an exact fixed condition at the pile tip, set yield_stress=inf for bottom.

Output

The output data has following structure:

output_directory
├── history # time history data (the same order as specified in input file)
│   ├── history_1.csv
│   ├── history_2.csv
│   ...
└── paraview # files to visualize waves in ParaView
    ├── fepile1d.pvd
    ├── fepile1d_1.vtu
    ...

Soil model

The soil resistance is decomposed into static and dynamic components:

\[\begin{aligned} R &= R_\mathrm{static}(u) + R_\mathrm{dynamic}(\dot{u}) \\ &= R_\mathrm{static}(u) + C \dot{u} \end{aligned}\]
